Stacks Utilizadas

TypeScript

Superset do JavaScript que adiciona tipagem estática ao código, ajudando a evitar erros e melhorando a produtividade.

React

Biblioteca para construção de interfaces de usuário, focada em componentes reutilizáveis e altamente interativos.

Next.js

Framework de React que permite a criação de sites e aplicações web rápidas e otimizadas para SEO, com renderização do lado do servidor.

Styled Components

Biblioteca para escrever estilos CSS diretamente em componentes React, permitindo um design mais organizado e modular.

Tailwind CSS

Framework CSS que fornece classes utilitárias para estilizar rapidamente interfaces com consistência e flexibilidade.
